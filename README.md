# BinaryHunt(EN)
BinaryHunt is a binary detection tool use as a Reverse Engineering Tool. The main goal of this tool is to detect modified code in files to their original.
I want this tool to be easy to use for any kind of person but mainly for security and learning. 
Please use my Project to make the world better.
Do not blame me if this project is a mess, i'm learning on my own.

Syntax >> Rad@rx-7600:~$ BinaryHunt -* your_file1  your_file2 

(Here all the commands -*);
        -f  --folder (sha256sum test on two identical folders).
        -s  --simple or --show (standard action of BinaryHunt(default) >> display changed addresses).
        -b  --binary (display the binary modified part).
        -a  --asembly (display the asm modified part).

If you want to add features that can be usefull let me know. <3
